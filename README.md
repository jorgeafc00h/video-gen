# Video Gen - LTX-2 Video Generation Project

AI-powered video creation toolkit with comprehensive guidelines for YouTube & Facebook content production using the LTX-2 model.

## Overview

This project provides best practices, strategies, and tooling for creating professional video content using LTX-2, an open-source audio-video foundation model that generates synchronized video and audio.
