## üí° Key Principles

### 1. **Volume Levels**
```
- Background music: -18dB to -22dB
- Narrator voice: -6dB to -10dB
- Sound effects: -12dB to -16dB
```

### 2. **Tempo for Relaxation**
```
- Very slow (sleepy): 40-55 BPM
- Slow (calm): 55-70 BPM
- Moderate (peaceful): 70-85 BPM
```

### 3. **Instrumentation**
**Best for bedtime:**
- Piano (soft, gentle)
- Harp (delicate, magical)
- Strings (warm, comforting)
- Flute (light, airy)
- Celesta (magical, dreamy)

**Avoid:**
- Percussion (too alerting)
- Brass (too bold)
- Fast tempos
- Sudden dynamic changes

### 4. **Integration with Narration**
```
Music should:
- Start 2 seconds before narration
- Duck slightly during speech (-3dB)
- Resume full volume after narration
- Continue seamlessly between scenes
```
